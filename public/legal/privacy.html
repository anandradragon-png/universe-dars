2. Какие данные мы собираем

При использовании Сервиса мы можем обрабатывать следующие данные:

  • Telegram ID, имя, фамилия, username (передаются автоматически через Telegram WebApp API)
  • Дата рождения (вводится пользователем для расчёта дара)
  • Имя, фамилия, пол, время и место рождения (вводятся пользователем в профиле, опционально)
  • Данные близких людей (имя, дата рождения, тип связи - вводятся пользователем)
  • Аватар (фото профиля, загружается пользователем)
  • История взаимодействий: результаты расчётов, диалоги с AI-наставником, ответы на квесты
  • Платёжные данные: идентификаторы транзакций (не номера карт - они обрабатываются платёжными системами)

3. Цели обработки

  • Предоставление основной функциональности Сервиса (расчёт дара, персональные послания)
  • Персонализация контента (послания Оракула, AI-коуч учитывают пол и возраст)
  • Обработка платежей и предоставление оплаченного доступа
  • Улучшение качества Сервиса
  • Связь с пользователем (уведомления о покупках, ответы на обращения)

4. Хранение данных

Персональные данные хранятся на серверах Supabase (PostgreSQL, регион EU). Данные защищены шифрованием и доступ к ним имеет только оператор Сервиса.

Часть данных кэшируется в localStorage браузера пользователя для обеспечения быстрой работы приложения.

Срок хранения: до удаления аккаунта пользователем или до прекращения работы Сервиса.

5. Передача данных третьим лицам

Мы не продаём и не передаём персональные данные третьим лицам, за исключением:

  • Telegram (через Telegram Bot API - для авторизации и платежей Stars)

  • Платёжные системы (Тинькофф, YupPay - для обработки платежей)
  • По требованию законодательства (в ответ на законный запрос уполномоченных органов)
