Настоящее Соглашение регулирует отношения между оператором сервиса YupDar (далее - Администрация) и пользователем Telegram Mini App @YupDarBot (далее - Пользователь).

Начиная использовать Сервис, Пользователь принимает условия настоящего Соглашения в полном объёме.

2. Описание сервиса

YupDar - это интерактивный сервис самопознания через авторскую систему 64 Даров. Сервис предоставляет:

  • Расчёт персонального дара по дате рождения
  • Персональные послания Оракула (с использованием AI)
  • Интерактивную Книгу Даров
  • AI-коуч для работы с гранями дара
  • Тренажёр интуиции
  • Расчёт совместимости
  • Персональную книгу для родителей о ребёнке

3. Уровни доступа

  • Странник (бесплатно) - базовый функционал: свой дар, 1 послание Оракула в день, тренажёр интуиции, превью Книги
  • Хранитель (платный) - полная Книга Даров, безлимит Оракула, 3 слота для близких, +50 кристаллов
  • Мастер (платный) - безлимит близких, расширенный AI-коуч, все функции
