ЛЕГЕНДЫ АТЛАНТИДЫ
Правила игры
🎯 Цель: набрать больше карт на руках, чем противник.
🎴 Колода и игроки
• 64 карты, каждая — один из даров-юпиков
• Играют трое: Ты, Противник и Высшие Силы (банк)
• На старте у каждого по 4 карты
💎 Параметры каждой карты
• 🔮 Магия — первая цифра кода
• 💚 Жизнь — вторая цифра
• ⚡ Сила — третья цифра
• 💥 Мощность — сумма всех трёх
▶ Ход игры
1. Ходящий выбирает параметр сражения
2. Все трое выкладывают карту закрытой рубашкой
3. Нажимаешь «Перевернуть» — карты раскрываются
4. У кого значение выбранного параметра выше — забирает все карты
5. Право следующего хода — у победителя
💥 Особый ход — Мощность
Каждый выкладывает 3 карты сразу. Считается сумма Мощности всех трёх. Если карт не хватает — Высшие добирают из колоды сразу на стол.
✨ Высшие Силы — банк
Не ходят как игрок, только докладывают карту в каждый раунд.
Если побеждают Высшие — все карты со стола уходят обратно в общую колоду, а право хода переходит к другому игроку.
⚖️ Ничья — спор
Если у двоих одинаковый максимум — они спорят: докладывают ещё по одной карте. Остальные наблюдают. Победитель спора забирает все карты со стола.
🔄 Обмен картами
Раз за партию можно обменять свою карту на случайную карту противника.
🎴 Добор
Между раундами Высшие добирают из колоды карты всем, у кого на руках меньше 4.
👑 Конец партии
Когда карт не остаётся для следующего раунда. Выигрывает тот из двоих (Ты или Противник), у кого больше карт на руках. Высшие в определении победителя не участвуют.
