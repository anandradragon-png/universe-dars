1. Общие положения

Настоящее Соглашение регулирует отношения между оператором сервиса YupDar (далее - Администрация) и пользователем Telegram Mini App @YupDarBot (далее - Пользователь).

Начиная использовать Сервис, Пользователь принимает условия настоящего Соглашения в полном объёме.

2. Описание сервиса

YupDar - это интерактивный сервис самопознания через авторскую систему 64 Даров. Сервис предоставляет:

  • Расчёт персонального дара по дате рождения
  • Персональные послания Оракула (с использованием AI)
  • Интерактивную Книгу Даров
  • AI-коуч для работы с гранями дара
  • Тренажёр интуиции
  • Расчёт совместимости
  • Персональную книгу для родителей о ребёнке

3. Тарифы и стоимость

Все цены указаны в рублях Российской Федерации.

  • Странник (бесплатно) — базовый функционал: расчёт своего дара, 1 послание Оракула в день, тренажёр интуиции, превью Книги Даров (10 глав)
  • Книга Даров (490 ₽, разовая покупка) — полная Книга Даров (94 главы) в интерактивном ридере + дизайнерская PDF-версия
  • Хранитель (990 ₽, разовая покупка) — всё из «Книги Даров» + безлимит Оракула + 3 слота для близких + AI-наставник + расчёт совместимости + 50 бонусных кристаллов мудрости
  • Мастер (990 ₽/месяц, ежемесячная подписка с автопродлением) — всё из «Хранителя» + безлимит AI-наставника + безлимит слотов + Дневник Дара с AI-анализом + приоритетная поддержка + ранний доступ к новым функциям

4. Оплата и возврат

Оплата принимается через: банковские карты Visa / Mastercard / МИР (через лицензированного платёжного провайдера), Telegram Stars, DarAI (NEAR Protocol).

Тарифы «Книга Даров» и «Хранитель» — разовые покупки, доступ бессрочный.

Тариф «Мастер» — ежемесячная подписка. С привязанной карты Покупателя автоматически списывается 990 ₽ каждый календарный месяц. Покупатель может отменить автопродление в любой момент в личном кабинете приложения — доступ сохранится до конца уже оплаченного периода.

Возврат средств возможен в течение 14 дней с момента покупки при обращении на yupdar@yandex.ru. Для Telegram Stars возврат осуществляется автоматически через Telegram. Возврат по банковской карте осуществляется на ту же карту в течение 10 рабочих дней.
